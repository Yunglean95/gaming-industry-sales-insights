## Project Overview
This project analyzes global gaming industry sales data to uncover trends, top-performing publishers, platforms, and regional sales insights. The analysis focuses on transforming raw data into meaningful business intelligence using interactive dashboards and visualizations.

---

## Objectives
- Analyze global video game sales performance
- Identify top publishers and platforms
- Understand regional sales distribution
- Generate actionable insights through data visualization
